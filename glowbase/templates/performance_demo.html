Query 3

Multi-Table JOIN

Products joined with brands and categories for high-rated items.

rating + brand_id
Execution Time {{ join_query_time }} ms
Rows Returned {{ join_query_count }} products
Indexes Used Rating / Brand 
SELECT p.product_name, b.brand_name, c.primary_category, p.rating
FROM products p
JOIN brands b ON p.brand_id = b.brand_id
JOIN product_categories pc ON p.product_id = pc.product_id
JOIN categories c ON pc.category_id = c.category_id
WHERE p.rating >= 4.5
ORDER BY p.rating DESC
LIMIT 50;

Index benefits: idx_products_rating supports the rating filter, while idx_products_brand_id supports the JOIN with the brands table.

When Indexes Help Most

  • WHERE clauses with =, >, <, and BETWEEN
  • JOIN conditions between related tables
  • ORDER BY on indexed columns
  • LIKE prefix searches, such as 'Cream%'
