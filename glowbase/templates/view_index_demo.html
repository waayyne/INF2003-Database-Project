SQL View: product_summary_view

This virtual table combines product and brand details so repeated product queries can use one cleaner source.

MariaDB view
View creation statement 
CREATE VIEW product_summary_view AS
SELECT
    p.product_id,
    p.product_name,
    b.brand_name,
    p.price_usd,
    p.rating,
    p.reviews,
    p.loves_count,
    p.out_of_stock
FROM products p, brands b
WHERE p.brand_id = b.brand_id;
Query using the view 
SELECT product_id, product_name, brand_name, price_usd, rating, reviews
FROM product_summary_view
WHERE rating >= 4.5
ORDER BY rating DESC
LIMIT 20;
