Products Above Average Rating

Find products rated higher than the overall average rating.

Nested Query 1
Show SQL 
SELECT p.product_id, p.product_name, b.brand_name, p.rating, p.price_usd
FROM products p, brands b
WHERE p.brand_id = b.brand_id
AND p.rating > (
    SELECT AVG(rating)
    FROM products
)
ORDER BY p.rating DESC
LIMIT 20;

Brands With More Than 3 Products

Show products where the brand appears in a nested aggregation.

Nested Query 2
Show SQL 
SELECT p.product_id, p.product_name, b.brand_name, p.rating, p.price_usd
FROM products p, brands b
WHERE p.brand_id = b.brand_id
AND p.brand_id IN (
    SELECT brand_id
    FROM products
    GROUP BY brand_id
    HAVING COUNT(*) > 3
)
ORDER BY b.brand_name, p.product_name
LIMIT 20;

Brands Above Overall Average Rating

Find brands whose average rating exceeds the overall average.

Nested Query 4
Show SQL 
SELECT b.brand_name, AVG(p.rating) AS avg_rating, COUNT(*) AS total_products
FROM products p, brands b
WHERE p.brand_id = b.brand_id
GROUP BY b.brand_name
HAVING AVG(p.rating) > (
    SELECT AVG(rating)
    FROM products
)
ORDER BY avg_rating DESC
LIMIT 20;

Highest Rated Products

Find products that share the highest rating in the dataset.

Nested Query 5
Show SQL 
SELECT p.product_id, p.product_name, b.brand_name, p.rating, p.price_usd
FROM products p, brands b
WHERE p.brand_id = b.brand_id
AND p.rating = (
    SELECT MAX(rating)
    FROM products

ORDER BY p.product_name

Brands With Out-of-Stock Products

Find products from brands that have at least one out-of-stock item.

Nested Query 6
Show SQL 
SELECT p.product_id, p.product_name, b.brand_name, p.out_of_stock
FROM products p, brands b
WHERE p.brand_id = b.brand_id
AND p.brand_id IN (
    SELECT brand_id
    FROM products
    WHERE out_of_stock = 1
)
ORDER BY b.brand_name, p.product_name
LIMIT 20;
